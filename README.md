# AutoStream AI Agent (Social-to-Lead Workflow)

## Overview

This project is an AutoStream AI Agent system that turns user conversations into leads.

It is different from chatbots.

The AutoStream AI Agent system does a lot of things.

It understands what the user wants.

It gets the product information.

It finds users who're really interested.

It collects data.

It does things on the backend.

---

## Architecture

The AutoStream AI Agent system works in a way.

It is like a workflow.

Here is how it works:

User Input goes to Intent Detection.

Then it goes to Decision Layer.

After that it goes to Knowledge Retrieval.

Finally it goes to Lead Capture.

### Key Components:

* Intent Detection is based on a kind of model.

* RAG Pipeline uses a vector search.

* State Management uses session memory.

* Tool Execution uses an API.

* Streamlit UI is used for the user interface.

## How to Run

To run the AutoStream AI Agent system you need to do a things.

First you need to clone the repository.

Then you need to go to the autostream-agent folder.

Next you need to make a virtual environment.

You need to activate the environment.

Then you need to install the requirements.

You need to add your API key to the.env file.

Finally you can run the app.

```bash

git clone <your-repo>

cd autostream-agent

python -m venv venv

pip install -r requirements.txt

# Add your API key in.env

python -m streamlit run app.py

```

## WhatsApp Integration

The AutoStream AI Agent system can be integrated with WhatsApp.

It uses the WhatsApp Business API.

It uses webhooks.

It uses an agent.

Here is how it works:

The user sends a message to WhatsApp.

The message goes to the webhook.

The webhook sends it to the AutoStream AI Agent system.

The AutoStream AI Agent system responds.

The response goes back, to the user.
